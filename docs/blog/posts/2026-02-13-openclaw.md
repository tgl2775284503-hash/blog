## **视频用到的命令**
- Node版本检查命令：
```
node -v
```
- Git版本检查命令：
```
git --version
```
- Windows开启脚本权限命令：
```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
- Windows权限检查命令：
```
Get-ExecutionPolicy -Scope CurrentUser
```
（如果显示RemoteSigned，就证明已开启成功）
- 安装Openclaw命令：
```
iwr -useb https://molt.bot/install.ps1 | iex
```
- Openclaw启动命令：
```
openclaw gateway
```
- 初始化node.js项目命令：
```
npm init -y
```
(如果如果提示缺少package.json时运行）
